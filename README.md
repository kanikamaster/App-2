## 🚀 セットアップ方法

1. リポジトリをクローン

```bash
git clone https://github.com/kanikamaster/App2.git
cd rauten-counter
```

２，依存パッケージをインストール

```bash
npm install
```

3 開発用サーバーを起動

```bash
npm run dev
```

4 ターミナルまたは表示されたURLをクリックしてブラウザで確認

```bash
http ://localhost:5173
```
